\doxysubsection{Подробное описание}
Инициализация и запуск приложения для визуализации данных о сне. 

Этот файл содержит функции для настройки окна GLFW и интеграции с Im\+Gui для рендеринга графического интерфейса. Приложение загружает данные о сне, анализирует их и представляет результаты в виде графического интерфейса.
